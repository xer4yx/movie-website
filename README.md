# movie-website

A sample movie website project using Vite and React

## Description

This is a project based from [Tech With Tim](https://www.youtube.com/@TechWithTim) on creating a website using
React. The project uses API to get a list of top movies being watched.

## Get Started

### How to run?
1. Go to `frontend` directory then install the dependencies by running this command:
```bash
    cd frontend
    npm install
```
